CCA - Assessment Coordinating Council

This is the CCA Web App which is an acronym for Assessment Coordinating Council of IPCA and it was appart of a pratical work for a class.

The tecnlologies used were C#, Javascript, Angular, Bootstrap, SQL and .NET

The main objective of this aplication was to create a platform and facilitate the evaluation of the university teachers.

The inicial idea was to implement a system capable of user authentication where only members of the council (role based) can create an evaluation. The data inserted in the web page form is written in a model excel and saved on a personal folder. Users can decide if they want to a pdf version of the submited evaluation.

Users can access all the submited evaluations and download them.

It was suggested at the end of the course to continue developing the project so that the university could use it. Some new features were added but in the end the university decided to drop the project.
